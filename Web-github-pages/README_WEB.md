# Music Visualizer - Web Edition

Transform your music into Nothing Phone glyph animations. Drag-drop your audio file, process it, and download the visualization data.

## Features

- **FFT Audio Analysis** - Real-time frequency analysis
- **Multiple Exports** - NGlyph (for GlyphModder), CSV, or Compact binary format
- **Zero Dependencies** - Uses native Web Audio API
- **Privacy First** - All processing happens in your browser

## How to Use

1. **Upload** - Drag an audio file (MP3, OGG, WAV, M4A)
2. **Select** - Choose your phone model
3. **Pick Format** - NGlyph, CSV, or Compact
4. **Process** - Click the button and wait
5. **Download** - Get your visualization file

Click the **?** button in the bottom-right for a full quick-start guide.

## Formats

- **NGlyph** - JSON format, use with GlyphModder to embed in audio
- **CSV** - Spreadsheet format for analysis
- **Compact** - Binary format, smallest file size

## Browser Support

Works on Chrome, Firefox, Safari 14+, and Edge.

## Next Steps
